SECTION 1  EXISTING SYSTEM (BEFORE THIS SESSION)


1. OCR Pipeline (ocr.py)
   Image  preprocess_prescription_image() (OpenCV)
         - Upscale if < 1600px
         - Deskew via Hough Lines
         - CLAHE contrast (clipLimit=2.0, tileGridSize=8x8)
         - Bilateral Filter de-noise
         - Unsharp Mask sharpening
         - Blue circular stamp removal (HSV mask)
    process_prescription_ocr() (Gemini 2.5 Flash Vision)
         - model: gemini-2.5-flash
         - temperature: 0.1
         - response_mime_type: application/json
         - Specialist Egyptian/Arabic-English prompt
         - Understands "slash notation": 1/8 = every 8 hours
         - Returns: patient, prescriber, medications list,
           confidence_score, interactions
    analyze_prescription_safety() (Groq llama-4-scout)
         - Dosing error detection
         - Drug interaction warnings
         - Frequency alerts with meal timing

   NOTE: OCR is NOT Tesseract or EasyOCR  it is Gemini Vision
         directly producing structured JSON output.

2. LLM Chat (chatbot.py)
   - BioMistral 7B running locally via Ollama
   - URL: http://localhost:11434
   - Default model: adrienbrault/biomistral-7b:Q4_K_M
   - Overridable via PHARMA_MODEL env variable
   - Chat format: Llama2 ChatML
     <s>[INST] <<SYS>>
     {system_prompt + RAG_context}
     <</SYS>>
     Clinical question: {user_message}
     [/INST]
   - Pre-LLM processing:
     - Abbreviation expansion: CKD -> chronic kidney disease
     - Clinical question detection (drug name regex)
     - RAG retrieval if clinical

3. RAG System (rag_engine.py)  PRESENT & WORKING
   knowledge_base/drugs.json
     -> _index_documents()
     -> Embeddings (priority):
          1. nomic-embed-text via Ollama
          2. ONNX all-MiniLM-L6 (fallback if Ollama offline)
     -> ChromaDB (local persistent at knowledge_base/chroma_db/)
   
   Query flow:
     query -> _expand_query() [brand->generic: plavix->clopidogrel]
           -> retrieve() [cosine similarity >= 0.35]
           -> TF-IDF fallback if Ollama offline
           -> chunks injected as "VERIFIED CLINICAL REFERENCES"
           -> BioMistral

SECTION 3  CHANGES MADE IN THIS SESSION (NEW FEATURES)


Feature 1: Drug Normalization (NEW FILE: drug_normalizer.py)

File: drug_normalizer.py (NEW  did not exist before)
Size: ~18KB | Catalog: 200+ brandgeneric entries

API:
  normalize(name: str) -> NormResult
  normalize_list(names: list[str]) -> list[NormResult]
  avg_confidence(results: list[NormResult]) -> float

NormResult dataclass:
  .generic        : INN/generic name
  .brand_matched  : matched brand name (if applicable)
  .confidence     : 0.01.0 confidence score
  .match_type     : 'exact'|'alias'|'fuzzy'|'generic'|'not_found'
  .notes          : human-readable explanation

Confidence tiers:
  exact match   = 0.97  (Panadol -> Paracetamol)
  generic name  = 0.95  (input is already a generic INN)
  alias/prefix  = 0.82  (prefix match >= 5 chars)
  fuzzy >= 0.78 = 0.500.91 (SequenceMatcher ratio-scaled)
  fuzzy >= 0.62 = 0.350.57 (low confidence, warn)
  not_found     = 0.30  (pass-through, name kept as-is)

Catalog categories (200+ entries):
  Antibiotics: Augmentin, Hi-biotic, Ciproxin, Flagyl,
               Zithromax, Klacid, Keflex, Zinnat, etc.
  GI: Colovatil, Antinal, Nexium, Losec, Motilium, etc.
  Analgesics: Panadol, Brufen, Voltaren, Celebrex, Tramal, etc.
  Cardiovascular: Norvasc, Concor, Zestril, Cozaar, Plavix,
                  Warfant, Lipitor, Crestor, Clexane, etc.
  Diabetes: Glucophage, Amaryl, Diamicron, Jardiance, Lantus, etc.
  Thyroid: Euthyrox, Eltroxin, Neomercazole, etc.
  Asthma/COPD: Ventolin, Seretide, Symbicort, Singulair, etc.
  Psychiatric/Neuro: Zoloft, Cipralex, Valium, Risperdal,
                     Tegretol, Lyrica, etc.
  Antihistamines: Clarityne, Zyrtec, Telfast, Phenergan, etc.
  Steroids: Medrol, Decadron, Betnovate, Bactroban, etc.
  Egyptian pediatric brands: V-Drop, Sanso Immune, Eubion,
                              Limotal, Kidssi, Ferose, etc.

Feature 2: Interaction Checker Upgrades (interaction_checker.py)

Changes to interaction_checker.py:

A) New function: _query_openfda(drug_a, drug_b) -> list[dict]
   - Queries: https://api.fda.gov/drug/label.json
   - Search: openfda.generic_name:"drug_a"
   - Checks if drug_b appears in drug_interactions text
   - timeout: 4 seconds
   - Returns interaction dict with source="openfda"
   - confidence: 0.72
   - Fails silently if no internet

B) 3-Tier interaction check in check_interactions():
   Tier 1: SQLite (pharma.db)  confidence: 0.88, source: "sqlite"
   Tier 2: Built-in _BUILTIN list  confidence: 0.92, source: "builtin"
   Tier 3: OpenFDA API  confidence: 0.72, source: "openfda"
   - Tier 3 only runs for pairs NOT already found in Tiers 1+2

C) format_interaction_alert()  source badge added to HTML:
   - sqlite  -> blue badge   (#E3F2FD / #1565C0) "Local DB XX%"
   - builtin -> purple badge (#EDE7F6 / #4527A0) "Evidence Base XX%"
   - openfda -> violet badge (#F3E5F5 / #6A1B9A) "OpenFDA XX%"
   - XX% = interaction_confidence * 100, shown in badge
